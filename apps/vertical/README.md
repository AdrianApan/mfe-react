# Vertical

## ℹ️ Requirements

Requires Node JS version `14.17.0+`. Using `nvm`:

```sh
nvm install 14.17.0
nvm use 14.17.0
```

## 🏃‍♀️ Running the app locally

#### _Installing dependencies:_

```
yarn install
```

#### _Running the app:_

```
yarn start
```
